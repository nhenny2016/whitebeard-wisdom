# Whitebeard Wisdom

A LuminaForge-level Next.js website for **Whitebeard Wisdom** — executive and life coaching for professionals ready for clarity, purpose, and lasting change.

Built with [LuminaForge Elite Standards](https://luminaforge.ai) (2026.05.28).

## Features

- Executive coaching & signature programs (CLARITY, DEPTH, LEGACY)
- Wisdom blog, workshops & speaking
- Free Wisdom Audit lead magnet
- Newsletter signup
- Booking calendar integration
- SEO, JSON-LD schema, FAQ, sitemap
- `/llms.txt` and `/llms-full.txt` for AI discoverability
- Vercel Analytics + Speed Insights

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy to Vercel and connect `whitebeardwisdom.com`.
